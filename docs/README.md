## 设计约束

文档中包含了明确的设计约束：
- 插件输出大小限制（1MB）
- 组件数量限制（50个/插件）
- 执行超时（30秒）
- 性能指标（Dashboard < 500ms 打开）
- 错误处理规范
- 安全底线约束

## 核心特性

1. **通用插件平台** - 支持任意 Python 脚本作为插件
2. **丰富的组件类型** - 进度条、列表、图表、文本、图片、交互式、自定义
3. **插件间数据共享** - 插件可读取其他插件的输出
4. **定时提醒/通知** - 插件可触发系统通知
5. **主题支持** - 深色/浅色/跟随系统
6. **组件排序和展开/折叠** - 用户可自定义显示

## 快速开始

1. 阅读 `PluginHub设计文档.md` 了解完整设计
2. 参考示例插件代码编写自己的插件
3. 按照实现计划分阶段开发

## 技术栈

- Swift 5.9+ (async/await)
- SwiftUI + AppKit
- macOS 13.0+
- Python 3.x (插件)
