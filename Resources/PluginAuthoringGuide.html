PluginHub 插件编写指南

PluginHub 使用 Python 脚本作为插件。脚本输出 JSON，PluginHub 将其渲染为 macOS 菜单栏组件。

快速开始

一个最简单的插件：

#!/usr/bin/env python3
# PluginHub:
#   name: 我的插件
#   description: 一个示例插件
#   icon: star.fill
# /PluginHub

import json
from datetime import datetime, timezone

data = {
    "updatedAt": datetime.now(timezone.utc).isoformat().replace("+00:00", "Z"),
    "title": "我的插件",
    "icon": "star.fill",
    "components": [
        {
            "type": "text",
            "data": {
                "id": "hello",
                "content": "Hello, PluginHub!",
                "style": "plain"
            }
        }
    ]
}

print(json.dumps(data, ensure_ascii=False))

元数据声明

在脚本开头用注释块声明插件信息：

# PluginHub:
#   name: 插件名称
#   description: 描述
#   author: 作者
#   version: 1.0.0
#   icon: SF Symbol 名称
#   parameters:
#     - name: key
#       type: string|secret|integer|boolean|choice
#       label: 显示名称
#       required: true|false
#       default: 默认值
#       options:
#         - label: 选项A
#           value: a
# /PluginHub

JSON 输出结构

{
  "updatedAt": "2024-01-01T00:00:00Z",  // ISO 8601 时间戳
  "title": "插件标题",
  "icon": "SF Symbol 名称",
  "badge": "角标文字",
  "notification": {                     // 可选：系统通知
    "title": "通知标题",
    "body": "通知内容",
    "sound": true,
    "scheduledAt": "2024-01-01T09:00:00Z"
  },
  "components": [...]                   // UI 组件列表
}

组件类型

progress（进度条）

{
  "type": "progress",
  "data": {
    "id": "cpu",
    "label": "CPU",
    "value": 45.5,
    "max": 100,
    "unit": "%",
    "color": "#007AFF",      // 可选 hex 颜色
    "style": "bar|ring|gauge"
  }
}

list（列表）

{
  "type": "list",
  "data": {
    "id": "info",
    "title": "系统信息",
    "style": "simple|detailed|table",
    "items": [
      {
        "title": "磁盘使用",
        "subtitle": "Macintosh HD",   // 仅 detailed 样式
        "value": "100GB / 512GB",
        "icon": "externaldrive",
        "color": "#34C759",
        "url": "https://..."           // 可点击链接
      }
    ]
  }
}

text（文本）

{
  "type": "text",
  "data": {
    "id": "status",
    "content": "一切正常",
    "style": "plain|alert|success|warning",
    "icon": "checkmark.circle"
  }
}

chart（图表）

{
  "type": "chart",
  "data": {
    "id": "trend",
    "title": "一周趋势",
    "type": "line|bar|pie",
    "xLabel": "日期",
    "yLabel": "数量",
    "data": [
      {"label": "周一", "value": 10, "series": "系列A"},
      {"label": "周二", "value": 15, "series": "系列A"}
    ]
  }
}

image（图片）

{
  "type": "image",
  "data": {
    "id": "photo",
    "url": "https://... 或 data:image/png;base64,...",
    "caption": "图片说明",
    "width": 300,
    "height": 200
  }
}

interactive（交互组件）

{
  "type": "interactive",
  "data": {
    "id": "scratchcard",
    "type": "scratchcard|button|input|toggle",
    "config": {
      "title": "每日刮刮乐",
      "description": "试试手气",
      "actions": [
        {"id": "scratch", "label": "刮开", "type": "callback"}
      ]
    },
    "state": {"prize": "恭喜中奖!"}
  }
}

custom（自定义）

{
  "type": "custom",
  "data": {
    "id": "stats",
    "type": "键值表",
    "data": {"key1": "value1", "key2": "value2"}
  }
}

环境变量

变量名说明
PLUGINHUB_SHARED_DIR插件间共享数据目录，可读取其他插件的输出
PLUGINHUB_TRIGGER_FILE写入此文件可立即触发刷新
PLUGINHUB_THERMAL_STATE系统热状态（nominal/fair/serious/critical）
PLUGINHUB_PARAM_*用户在设置中配置的参数值

约束限制

  • JSON 输出最大 1MB
  • 单个插件最多 50 个组件
  • 执行超时 30 秒
  • 刷新间隔最小 10 秒
